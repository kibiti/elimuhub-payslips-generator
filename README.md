# Elimuhub Payslips Generator

A lightweight, client-side Weekly Tutor Payslip Generator for Elimuhub Education Consultants — ready to publish on GitHub Pages.

## Features
- Input payslip details and preview a polished payslip
- Automatic calculations (gross, deductions, net pay)
- Print or save as PDF via browser print dialog
- Responsive layout for desktop and mobile
- Clean separation of HTML / CSS / JS for easy maintenance
